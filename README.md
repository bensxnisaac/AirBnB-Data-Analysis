# NYC Airbnb Market Analysis

## Project Overview

This project analyzes Airbnb listings in New York City to uncover pricing patterns,
market distribution, and key factors influencing listing performance.

The objective is to derive actionable insights for hosts and potential investors.

---

## Business Questions

- Which borough dominates Airbnb listings?
- What factors influence listing price?
- How does room type affect pricing?
- Do cheaper listings receive more reviews?
- Which borough generates the highest potential revenue?

---

## 🛠 Tools & Technologies

- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

---

## Data Cleaning

- Removed duplicate records
- Filled missing host names with "Unknown"
- Converted date columns to datetime format
- Handled missing review metrics
- Checked for price outliers

---

## Key Insights

- Manhattan has the highest average price and highest number of listings.
- Entire homes command premium pricing.
- Price distribution is heavily right-skewed.
- Moderate-priced listings receive more reviews.

---

## Business Recommendations

- Investors seeking premium returns should focus on Manhattan.
- Mid-range listings may offer the best balance of demand and pricing.
- Hosts in Brooklyn can leverage high listing density to optimize competitive pricing.
